Folder structure:
create a well maintainable and readable folder structure.

articul8/
├── .next/                     # Generated by Next.js during build
├── public/                    # Public assets like images, fonts, etc.
│   ├── favicon.ico            # Website favicon
│   └── images/                # Images folder for public assets
├── app/                       # App directory for Next.js app-based routing
│   ├── (home)/                # Home feature, folder-based routing
│   │   ├── Hero.tsx           # Hero section of homepage
│   │   ├── Features.tsx       # Features section of homepage
│   │   ├── BlogPreview.tsx    # Blog preview section of homepage
│   │   ├── Contact.tsx        # Contact section of homepage
│   │   └── page.tsx           # Main landing page component for home
│   ├── about/                 # About page feature
│   │   └── page.tsx           # About page component
│   ├── blog/                  # Blog feature
│   │   ├── [slug]/            # Dynamic routes for individual blog posts
│   │   │   └── page.tsx       # Blog post page component (dynamic slug)
│   │   └── page.tsx           # Blog listing page
│   ├── contact-us/            # Contact Us page feature
│   │   └── page.tsx           # Contact Us page component
│   ├── layout.tsx             # Root layout for the app
│   └── page.tsx               # Fallback or secondary home page if needed
├── components/                # Reusable components (shared across pages)
│   ├── Header.tsx             # Header component for site navigation
│   ├── Footer.tsx             # Footer component
│   ├── Seo.tsx                # SEO component for handling meta tags
│   └── Button.tsx             # Example reusable button component
├── styles/                    # Global styles
│   └── globals.css            # Main global CSS file
├── utils/                     # Utility functions and helpers
│   └── api.ts                 # API helper functions for Strapi integration
├── .env                       # Environment variables (e.g., Strapi credentials)
├── next.config.js             # Next.js configuration file
├── tsconfig.json              # TypeScript configuration file
└── package.json               # Project dependencies and scripts
